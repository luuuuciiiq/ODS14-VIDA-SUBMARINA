# ODS14-VIDA-SUBMARINA


Créditos:
Fotografía - https://paulnicklen.com/
Información ODS nro. 14 - https://www.un.org/sustainabledevelopment/es/oceans/

Enfoque conceptual: 
Para esta página web busqué mostrar la belleza de la vida submarina mediante imágenes cautivadoras, que apelen al espectador y permitan la reflexión sobre la importancia del cuidado del planeta Tierra y los animales. La información en la página menciona la vitalidad de la preservación de nuestros recursos y las consecuencias del mal uso de los mismos,  tales como el calentamiento global, pesca excesiva,  y contaminación abundante.
